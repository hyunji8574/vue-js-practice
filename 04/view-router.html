뷰 라우터

라우팅이란 웹 페이지간 이동 방법을 말한다. 주로 싱글 페이지 앱에서 사용한다.
라우팅을 이용하면 화면간 전환이 매끄러워져 UX를 향상시킬수 있다.

*싱글 페이지 앱 : 페이지를 이동할때마다 서버에 요청하지 않고 라우팅을 이용해 화면만 갱신시키는 패턴을 적용한 앱
뷰 라우터 예제

Main컴포넌트로 이동 Login컴포넌트로 이동 


        var Main = {
            template: "<div>main</div>"
        };
        var Login = {
            template: "<div>login</div>"
        };

        var routes = [
            { path: '/main', component: Main },
            { path: '/login', component: Login }
        ]

        const router = new VueRouter({
        routes 
        });

        new Vue({
            router
        }).$mount('.app');
